# Secure User Authentication System - Task 1

This is a backend project for secure user authentication developed as part of my internship task.

## Features
- User registration
- User login
- Password validation and authentication
- Secure backend API

## Project Files
- server.js - Main server file
- auth.js - Authentication logic
- user.js - User model or schema
- package.json - Project dependencies

## Database
- Local MongoDB (MongoDB Compass)

## How to Run
1. Install dependencies: npm install  
2. Start server: node server.js  

## Note
This project is fully backend-based and uses Node.js with MongoDB for authentication.
